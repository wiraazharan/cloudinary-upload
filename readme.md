# CLOUDINARY-UPLOAD

### Setup
RUN : 
```sh
$ composer install
```


### Create File with titled "cloud_inary.php" and put inside config folder

File contains :
```sh
<?php
return [
    'cloud_name' => 'xxx',
    'api_key' => 'xxx',
    'api_secret' => 'xxx',
    'base_secure_url' => 'https://res.cloudinary.com/',
];
```
